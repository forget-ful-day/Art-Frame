# Мое Портфолио и Магазин

Статический сайт-портфолио с каталогом товаров.

## Структура проекта

- `index.html` - Главная страница с информацией обо мне
- `about.html` - Детальная информация о моей деятельности
- `products.html` - Страница с каталогом товаров
- `css/style.css` - Стили сайта
- `js/script.js` - JavaScript для функциональности
- `images/` - Папка с изображениями

## Настройка

1. Замените placeholder-текст в HTML файлах на свою информацию
2. Добавьте свои изображения в папку `images/`
3. Обновите данные товаров в `js/script.js`
4. Укажите правильные ссылки для кнопок "Купить"

## Загрузка на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы в репозиторий
3. В настройках репозитория включите GitHub Pages
4. Выберите ветку `main` как источник

## Локальный запуск

Просто откройте `index.html` в браузере или используйте Live Server в VS Code.
